EmployeeController.cs - Controller Employes

COUCHE 4 : CONTROLLERS • CRUD employes avec auth

Role — Endpoints REST pour gerer les employes avec authentification requise.

Endpoints

Methode URL Action Auth
GET /api/employee Liste tous Authentifie
GET /api/employee/{id} Details Authentifie
POST /api/employee Creer Authentifie
PUT /api/employee/{id} Modifier Authentifie
DELETE /api/employee/{id} Supprimer Manager/Admin

Points Cles

Create — Retourne 201 Created avec CreatedAtAction. Header Location pointe vers le nouvel employe.
Delete — Protection avec [Authorize(Policy = "ManagerOrAdmin")]. Seuls Manager et Admin peuvent supprimer.
